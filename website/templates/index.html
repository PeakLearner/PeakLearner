PeakLearner: Interactive Machine Learning for Genome Analysis

PeakLearner is a fast and simple genome analysis and labeling tool with the goal of providing researchers the ability to add labels and generate models for ChIP-seq data. It is built upon Jbrowse, a fast genome browser. For how to use PeakLearner, please see the tutorial page, or the video tutorials (TODO).

Features

  • Ease of labeling
  • Group Collaboration (Soon)
  • Automatic Model Generation
